Wireframe

Wireframe é uma apresentação visual da estrutura basica de uma pagina. ela deve ser feita de maneira mais simples possivel, mostrando apenas o essencial, como uma especie de rascuho, sem cores ou imagens. Clique no botão QR code para obter o wireframe desta pagina
